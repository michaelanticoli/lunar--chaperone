I. INTERIOR FORMAT SPECIFICATIONS
(InDesign Blueprint — Print + Interactive PDF Compatible)
Trim Size Recommendation
Option A (Most Premium Feel):
6 x 9 inches
Elegant, bookish, intimate.
Option B (Workbook Feel, Slightly More Modern):
7 x 10 inches
For $20 print, I recommend 6 x 9.
It feels curated, not mass-market.

Bleed
0.125 in (3mm) on all sides.
Interior will use minimal full bleed — mostly white space.

Margins (6 x 9 Format)
Top: 0.9 in
Bottom: 1 in
Inside: 0.9 in
Outside: 1 in
Why:
Slightly wider outer margin creates breathing room.
Slightly heavier bottom margin anchors the page visually.

Baseline Grid
Body text: 11.5 pt
Leading: 17 pt
Set baseline grid to 17 pt.
Align all body copy to baseline grid.
Pull quotes may float off grid intentionally.

Column Structure
Single column layout for 90% of book.
Column width: approx. 3.75–4 inches.
Do not go full width — that destroys luxury pacing.

Master Page System
You will create 5 Master Pages:

A — Section Opener
• Title centered vertically in upper third
• Small caps serif
• 2pt thin divider line beneath
• Minimal illustration bottom third
• No running header

B — Ritual Page
• Narrow centered text block
• Pull quote styled separately
• Illustration bottom corner
• Half page blank journaling space

C — Integration Page
• Large centered keyword top
• Three concise practices
• 1/3 page blank
• Bottom micro footer

D — Full Illustration Spread
• Left page illustration dominant
• Right page poetic script narrow column

E — Journal Page
• Subtle header
• 70% blank space
• Optional faint dotted writing guides (10% opacity)

Running Footer
Centered, very small:
MOONtuner — Pisces → Libra
© 2026
8pt sans serif.
Never intrusive.

II. TYPOGRAPHY SYSTEM
Keep it disciplined.
Serif (Headings)
High contrast modern serif.
Examples to consider:
• Canela
• Cormorant Garamond (premium styling)
• Editorial New
• Noe Display
Tracking: +40 to +60 for small caps.

Sans (Body)
Clean, neutral.
• Suisse Intl
• Avenir
• Neue Haas Grotesk
• Inter (budget-friendly)
Tracking: +10 to +20.

Hierarchy Scale
Section Title: 34–40 pt
Subsection: 18 pt
Pull Quote: 22–26 pt
Body: 11.5 pt
Footer: 8 pt
Never use bold for emphasis.
Use space instead.
